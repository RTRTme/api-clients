tcp-sync.php: 
==============================================================================

DESCRIPTION
===========
Basic Demo of how to retrieve RTRT.me data via the TCP and Web API protocol

REQUIRES
============
php 5.26+
php-cli
php-curl (for web-api)

USAGE
============
Change CONFIGURATION section as needed then run as php cli script.

    php tcp-api-demo.php  (NOTE: TCP protocol requires that we open ports for your IP)
or
    php web-api-demo.php


DOCUMENTATION
=============
General documentation for RTRT.me is available at:

    http://rtrt.me/docs
